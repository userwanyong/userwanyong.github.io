## 1. 项目介绍
《灵犀助手》一个集成智能问答、日志分析、发帖通知、RAG问答、模拟器等功能的AI智能助手
## 2. 技术栈
- Springboot
- Mysql
- Pgvector
- Redis
- Elasticsearch
- Logstash
- Kibana
