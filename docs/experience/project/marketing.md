## 1. 项目介绍
《幸运补给站》一个整合 抽奖/积分/兑换/返利/广播 等功能的平台
## 2. 技术栈
- Springboot
- Mysql
- Redis
- Elasticsearch
- Mybatis-Plus
- Canal
- Rabbitmq
- Zookeeper
- Xxl-job
- Prometheus
- Grafana
